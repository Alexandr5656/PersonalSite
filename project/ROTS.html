Project title

This project was thought of during my covid winter break. I wanted to teach my cousin's in Florida how to program in a fun way. I decided the most fun way was for them to learn programming fundemental by testing out their programs on a robot and seeing it move. The only issue was that they were more than a thousand miles away for me. So I decided to create a website that they could upload their code to and watch the robot run their code in real time. If multiple programs are uploaded then itll be run in order of first in first out.

The website is built using basic html/CSS and some javascript. It's running on a Ubuntu server that is set up on a RIT organization's server rack. To communicate with the robot the website has php scripts for when code is being uploaded and for when the next program is ready to be ran on the robot.

The robot was based off a Raspberry Pi robot kit. On the raspberry pi a program is cosntantly scanning the website for files to run. Once the website says theres a program waiting the robot will download the first program in the queue and then automatically run it. Once it runs it course the robot pings the website for any other programs.

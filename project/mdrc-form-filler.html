MDRC Form Filler

At the start of the fall 2021 school semester I was elected the Multi-Disciplinary Robotics Club Co-Treasurer. Part of this job was filling out purchase request pdfs for every single item that we wanted to purchase. We also had to make new ones if the items were from different companies. These purchase forms took hours to complete by hand and there was no program to automate it. A few years back someone in the organization made a slack bot to automate these forms but it has since been forgotten.

This python script takes in data from a downloaded excel sheet. Afterwards it parses the sheet into different objects first based on company that the item is being bought from then grouped together in groups of five as the sheet only fits five items. I also included a option to have it sort by team instead. After all the items were on the sheet the bot would then go through all the webpages for the items and add them into a cart prompting the user to enter the shipping costs. Once shipping costs were put in all other details were filled in and made the forms ready to be sent out to our advisor for approval.

The current future for this project is to make the job of future treasurers easier. First I want to make it easier to get the sheets to the program easier using their api. I also want to make it change values on the sheets automatically when the forms are made. Next I'd want to make a UI so the future treasurer can have a simplier time visualizing the program even if they dont know programming. Lastly I would want to create a slack bot/webUI that makes it easier to add items to the purchase request sheet.
